WeatherNow

WeatherNow is a modern web application that delivers real-time weather updates through a clean and responsive design. It provides accurate three-day forecasts along with detailed hourly predictions, helping users plan their day with confidence. The application enhances clarity by displaying precise location information including the city, region, and country. With its glassmorphism-inspired interface, WeatherNow combines reliability with a modern and engaging user experience.

Features

  • Real-time weather updates
  • Hourly & 3-day forecast
  • City-based search
  • Favorites for quick access
  • Glassmorphism UI & gradient theme

Technology Used

  • Frontend: HTML, CSS, JavaScript
  • Styling: Responsive Design, Flexbox/Grid
  • API: WeatherAPI

Purpose

This project demonstrates my ability to build interactive web apps, integrate third-party APIs, and implement modern UI/UX. It highlights responsive layouts, state handling, and user-friendly design.
